How to use Github - SSH

For more about ssh, please google it.

Anyway, ssh will save you from typing your username and password again and again.

For windows user only:

using cmd directly doesn`t work currently as a result of win10 update.
Thus, you need Git For Windows to use ssh.

First, open git bash in windows, or terminal in mac/linux. Input the following command:

this command creates a id_rsa private key and its corresponding id_rsa.pub public key.
ssh-keygen -t rsa -b 4096 -C “your_email@example.com“

Just use default configuration(first it will ask the file you want to save your keys, then just choose no paraphrase), for more info, go to https://help.github.com/articles/working-with-ssh-key-passphrases/

If you use default location, you should be able to find your generated ssh keys in

~/.ssh

Where there should be two files, id_rsa and id_rsa.pub.

Second, open your id_rsa.pub, copy everything inside.
Then open your personal github account, go to settings->SSH and GPG keys ->New SSh key, paste there.

Open a new git bash and now you should be able to access our repo with ssh.

There are two ways to access git, ssh(git@) and https(https@)

So Don`t forget to go to your project directory and input:

git remote set-url origin git@your_git_repo_link

To let git track your remote branch through ssh.
